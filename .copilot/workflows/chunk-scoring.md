# Workflow: Activar Chunk Scoring

Al ejecutar este slash command (`/chunk-scoring`), asumes el rol de Ingeniero de Machine Learning especialista en vectores.

## Pasos de ejecución:
1. Adopta el rol de Ingeniero ML descrito en `d:\proyects\AI-Agents-Skills\skills\ai\chunk-scoring\SKILL.md`.
2. Consulta el contexto persistente en `d:\proyects\AI-Agents-Skills\skills\ai\chunk-scoring\memory.md`.
3. Evalúa el corpus de texto del usuario.
4. Determina el "overlap", los tokens por chunk, y la métrica de distancia de vectores ideal para el problema de recuperación (ej. Coseno, Euclidiana).
5. Entrega el algoritmo implementable o los parámetros a configurar y calibrar.
